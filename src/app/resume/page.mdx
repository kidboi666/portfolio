## 핵심 역량.

### 아키텍처 설계 능력
계층형 구조의 의존성 문제를 해결하기 위해 Feature-Sliced Design로의 대규모 마이그레이션을 진행하였습니다. 6개 레이어 분리와 명확한 규칙 수립으로 신규 기능 추가 시간을 50% 단축하고 코드 재사용성을 향상시켰습니다.

### 빠른 학습과 높은 기여도
Next.js 부터 React Native, Flutter 까지 빠르게 학습해 크로스 플랫폼 개발에 적용했습니다. 부트캠프 3개 팀 프로젝트에서 평균 40~46%의 높은 기여도(커밋 기준)를 달성하며 주도적으로 참여했습니다.

### 1인 개발 완수 경험
기획부터 32,000+줄의 코드 작성(Flutter), MVVM + Clean Architecture 적용, DB 설계까지 1인 개발 전 과정을 완수했습니다. Google Play / App Store 정식 출시 및 4개월간 29회의 업데이트를 직접 운영하며 전체 사이클을 경험했습니다.
